已成功为“工作助手”（`work-assistant`）和“生活助手”（`life-assistant`）开启记忆隔离。

**配置详情：**
- **工作助手 (`work-assistant`)**：记忆隔离已开启 (`isolation_agent: true`)
- **生活助手 (`life-assistant`)**：记忆隔离已开启 (`isolation_agent: true`)

**这意味着：**
1. **记忆独立存储**：工作助手中产生的记忆（如项目决策、技术偏好）将标记为 `agent_id: work-assistant`，而生活助手的记忆（如旅行计划、个人喜好）将标记为 `agent_id: life-assistant`。
2. **互不干扰**：当你在工作助手中询问信息时，它只会检索与工作相关的记忆；在生活助手中则只检索生活相关的记忆。
3. **跨助手切换**：如果你在不同的助手客户端之间切换，只要它们正确使用了各自的 `agent_id`，记忆就会自动隔离。

**后续建议：**
- 确保你在调用“工作助手”时，其内部配置或启动参数中使用的 `agent_id` 确实是 `work-assistant`。
- 同样，确保“生活助手”使用 `life-assistant` 作为其 `agent_id`。
- 如果未来需要关闭隔离或修改 ID，可以随时告诉我。
